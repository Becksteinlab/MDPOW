=========
Workflows
=========

.. _workflows:

.. versionadded:: 0.9.0

MDPOW analysis workflows provide automation functions for use with
analysis :ref:`tools` based on the :ref:`framework`. Existing workflow
modules, :mod:`~mdpow.workflows.dihedrals`, include functions
for use with :class:`~mdpow.analysis.dihedral.DihedralAnalysis`, and a
:mod:`~mdpow.workflows.base` module for use with
existing and upcoming workflows.
